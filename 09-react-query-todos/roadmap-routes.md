# Better Todos Roadmap Routes

| url             | description                |
|-----------------|----------------------------|
| `/`             | Welcome page               |
| `/todos`        | List of all todos          |
| `/todos/1`      | Show single todo with ID 1 |
| `/todos/2`      | Show single todo with ID 2 |
| `/todos/add`    | Show create todo form      |
| `/todos/1/edit` | Show edit todo form        |
